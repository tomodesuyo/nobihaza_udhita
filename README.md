# 説明
のびハザをウディタに移植しています。
一応Windows環境なら誰でも改造できるはずです。
# 改造方法
https://www.silversecond.com/WolfRPGEditor/<br>
↑のページからウディタのプログラムのみをダウンロードして、ディレクトリに入れてください。
プロ版ではない場合、混同しないようにGamePro.exeを削除したほうがいいです。
Editor.exe(プロ版の場合はEditorPro.exe)を起動すれば改造できます。<br>
